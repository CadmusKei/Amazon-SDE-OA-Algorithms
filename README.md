# CIQ-Java

## Overview

CIQ-Java (Common Interview Questions – Java) is a repository dedicated to practicing common algorithmic and problem-solving patterns frequently encountered in software engineering interviews.

The goal of this project is to strengthen my Java fundamentals, improve algorithm design intuition, and prepare for technical assessments such as online coding assessments and live interviews.

## Repository Structure

Each Java file focuses on a specific problem pattern or topic, similar to structured practice platforms.

Examples of topics include:

* Sliding Window
* Counting
* PREFIX sum    
* Grid Methods
* Greedy Algorithms

A dedicated `sketch.java` file is included for experimentation and practice.

## Purpose

This repository serves as a personal algorithm dictionary where I:

* Document problem-solving strategies
* Implement clean Java solutions
* Track learning progress over time
